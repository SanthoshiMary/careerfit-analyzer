About CareerFit Analyzer

CareerFit Analyzer is a smart resume analysis system that compares a candidate’s resume with a target job description. It identifies matched skills, missing skills, ATS readiness, and provides suggestions, learning roadmap, and interview preparation guidance.

Core Features

  • Resume and Job Description Matching
  • Skill Gap Detection
  • ATS Readiness Evaluation
  • Learning Roadmap Generation
  • Role-Based Interview Questions
  • Semantic and Taxonomy-Based Matching

Technologies Used

  • Frontend: HTML, CSS, JavaScript
  • Backend: Python Flask
  • Database: SQLite
  • Resume Parsing: pdfplumber, python-docx
